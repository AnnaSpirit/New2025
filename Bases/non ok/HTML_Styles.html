The HTML style attribute is used to add styles to an element, such as color, font, size, and more.


style="property:value;": The property is a CSS property. The value is a CSS value.

  • Background Color
    • style="background-color: *Color code or name;"
  • Text Color
    • style="color: *Color code or name;"
  • Fonts
    • style="font-family:name of font;"
  • Text Size
    • style="font-size:300%;"
  • Text Alignment
    • style="text-align:left;"
      • Aligne le texte à la gauche. C'est la valeur initiale de cette propriété.
    • style="text-align:right;"
      • Aligne le texte à la droite.
    • style="text-align:center;"
      • Aligne le texte au centre.
    • style="text-align:justify;"
      • Étend les lignes pour que chaque ligne ait la même largeur.
    • style="text-align:initial;"
      • Fait utiliser la valeur initiale.
    • style="text-align:inherit;"
      • Hérite la propriété de son élément parent.
