CSS Selectors

5 categories de Sélecteurs en CSS

Simple

Sélection des éléments basés sur:

  • leur nom (ex: h3),
  • id (introduit par un dièse (#)),
  • class (introduit par un point(.)).

Combinés

Sélection des éléments basés sur une relation spécifique entre eux, séparés par une virgule (,).

Pseudo-class

Sélection des éléments basés sur un état donné

  • hover

    • est utilisé pour définir le comportement d'un élément lorsqu'un curseur de souris est placé dessus.

  • active

    • Cible un élément lorsqu'il est activé, généralement lorsqu'il est cliqué.

  • focus

    • Cible un élément lorsqu'il a le focus, par exemple, après avoir été sélectionné avec la tabulation.

  • visited

    • Cible un lien hypertexte qui a déjà été visité par l'utilisateur.

  • first-child

    • Cible le premier enfant d'un élément parent.

  • last-child

    • Cible le dernier enfant d'un élément parent.

  • nth-child()

    • Cible un élément en fonction de sa position dans l'ordre des enfants de son parent, en utilisant une formule comme "2n" pour cibler chaque deuxième enfant.

  • nth-of-type()

    • Similaire à :nth-child(), mais cible uniquement les éléments du même type.

  • not()

    • Permet de sélectionner tous les éléments qui ne correspondent pas à un sélecteur spécifié.

  • checked

    • Cible un élément de type input qui est coché (par exemple, une case à cocher).

Pseudo-elements

Ciblent une certaine partie d'un élément plutôt que l'élément tout entier. Ils sont représentés par des double-points (::) après le sélecteur CSS.

  • ::before

    • Permet d'insérer du contenu avant le contenu réel d'un élément

  • ::after

    • Permet d'insérer du contenu après le contenu réel

  • ::first-line

    • Permet de cibler la première ligne de texte à l'intérieur d'un élément

  • ::first-letter

    • Permet de cibler la première lettre d'un élément.

  • ::selection

    • Permet de cibler le texte sélectionné par l'utilisateur.

  • ::placeholder

    • Permet de cibler le texte de l'attribut placeholder d'un élément de formulaire (par exemple, un champ de saisie).

  • ::marker

    • Permet de cibler le marqueur d'une liste (par exemple, les puces d'une liste à puce).

Attribute

(sélection sur la présence d'un attribut avec une valeur bien précise) EX: a[title] { }

Combinateur

Universel (*) EX: * { }

Combinateur Enfant (>) EX: article > p { }

Combinateur parent ( ) EX: article p { }

element, element
écrire la suite

Combinateur de voisin direct (FRERES) (sélectionner un élément uniquement si celui-ci « suit » un élément donné et que les deux éléments sont les fils d un même élément parent) EX: premier_element + element_cible { }

Combinateur de voisin généraux (Les deux éléments sont des fils d'un même parent Element, voisins immédiats ou pas) EX: premier_element ~ second_element { }

CSS propriété

Background color ou image

Borders border-style, border-width, border-color

styles de mise en page du texte : indique le nombre d'espaces à ménager horizontalement avant le début de la première ligne de texte du contenu. : définit la façon de signaler aux utilisateurs le contenu en débordement (et donc non affiché). : définit comment les espaces blancs et les sauts de ligne associés dans l'élément sont gérés. : spécifie s'il y a césure dans les mots. : définit la direction du texte (cela dépend de la langue et généralement, il vaut mieux laisser HTML la gérer car elle est liée au contenu du texte.) : détend ou renforce la rupture de ligne pour les langues asiatiques. : définit comment la dernière ligne d'un bloc ou d'une ligne, juste avant un saut de ligne forcé, est alignée. text-orientation : définit l'orientation du texte dans une ligne. word-wrap : indique si le navigateur peut replier une ligne en conservant l'ordre des mots pour empêcher un débordement. writing-mode : définit si les lignes de texte sont disposées horizontalement ou verticalement et la direction des lignes suivantes.
